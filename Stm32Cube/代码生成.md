# 时钟配置
* HSE：高速外部时钟 
* LSE：低速外部时钟 --可以使用外部rtc做时钟源，需要开发板

# ide代码生成
* 主界面-project manager -> code generator -> generator h.cfile
* 快捷操作
* ctrl+s       保存并生成代码文件
* ctrl+b       编译代码
* ctrl+i       修正缩进
* ctrl+q       代码回到上次修改处
* alt+left     代码回到上一级
* alt_right    代码回到下一级
* ctrl+shit+f  修正格式
* ctrl+shit+/  块注释
* ctrl+shit+\  取消块注释
* ctrl+shit+b  打/取消断点
* ctrl+7       注释
* ctrl+tab     切换c/h文件
* F2           查看滚动框元素
* F3           跳到定义生成位置
* alt+enter    打开属性对话框

# 添加文件
* alt+enter 打开设置 c/c++ General --> Paths and Symbols
* 在Includes添加头文件路径 --> source location添加程序目录节点

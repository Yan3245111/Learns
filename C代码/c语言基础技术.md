1: int *p; 和 int **p; 的区别
示例：
int a = 10;
int b = 20;

int *p = &a;   // 不可写成 *p = a; p指向的是a的地址，*p指向a的值，如果*p改变，则a的值改变
int **p1 = &p; // 不可写成 **p = p || **p = *p; *p 指向p的地址，即a的地址，**p指向p的值，即a的值

**p1 一般用于处理动态二维数组
例如：
int **p1 = (int **)malloc(3 * sizeof(int *));
for (int i = 0; i < 3; i++) {
    p1[i] = (int *)malloc(3 * sizeof(int));
}

2： const int *p; int *const p; const int *const p; 使用的时候必须初始化
声明方式	解释
const int *p;	p 是一个指向 const int 的指针，即 p 可以改变，但 *p 不能改变。
int *const p;	p 是一个 const 指针，即 p 不能改变（指针本身不可修改），但 *p 可以改变。
const int *const p;	p 是一个 const 指针，指向 const int，即 p 不能改变，*p 也不能改变。
示例
int a = 1;
int b = 2;
const int *p = &a;
p = &b;  --√
*p = 3;  --❌

int *const p1 = &a;
p1 = &b;  --❌
*p1 = 20; --√

const int *const p2 = &a;
值和地址都不可改变


3：野指针
当你释放了一块内存（例如使用 free() 或 delete），但指针仍然指向这块已释放的内存，指针就成了野指针。
int *p = (int *)malloc(sizeof(int));
*p = 10;
free(p);'
p = NULL; 即可，如果不加这个就是野指针

4：悬空指针
离开作用域，访问指针的地址，例如在函数内声明int a = 1; int *p = &a; 但是在函数外 重新定义 int *p; 然后访问p


5：字节对齐
#pragma pack(push, n)
#pragma pack(pop)

优点：使用1字节对齐，可以减少结构体的内存占用
缺点：对于某些硬件可能会影响效率，比如64位处理器是8字节的，如果使用2字节对齐，可能会降低效率

6：位域（Bit Field）
不能为负数，一般在结构体中指定变量位数使用
优点：压缩数据存储，将多个小数据压缩到一个int里，即32位


7：RTOS操作系统
优先级调度：
按照优先级执行任务，先优先最高优先级进程执行
如果优先级相同，就是谁先来谁执行
分抢占式和非抢占式
抢占式：如果新执行的任务优先级高于现在执行的任务，则先执行新任务
非抢占式：要等待现在的任务执行完毕，才能执行新的优先级高的任务
时间片轮询：
按照FIFO(先来先执行)的顺序排队
进程执行一个时间片，如果没有完成，则排到最后，等待下一轮执行

线程间通信： FreeRTOS/RT-Thread都有这几个方法
信号：rt_sem_t
互斥量：rt_mutex_t
事件：rt_event_t
队列：rt_mq_t  先进先出，异步消息传递，可以在不同线程间通信，支持阻塞与非阻塞

10：串口
同步：发送和接收设备的时钟信号是同步的，使用外部时钟线传输，速度更快
异步：不需要共享时钟信号，通过设置波特率，起始位，奇偶校验，数据位，停止位，根据设定的波特率时序判断开始和结束

11：I2C
SCL和SDA传输数据，主机->从机通信方式，允许多个设备通过同一条总线进行通信
主机：负责生成SCL时钟信号，可以发起数据传输
从机：响应主机命令，不生成时钟信号，也不能主动发起通信
I2C地址是7位，所以最后128个从机设备，但是0x00（保留地址）0x7f（广播地址），所以一般只有126个

12：SPI
MISO: 主输入，从输出
MOSI: 主输出，从输入
SCK：时钟信号
SS：片选
波特率= 时钟频率/（16 * 分频系数） 例：10 * 1e6(Hz) / (16 * 104) = 9600

13: CAN
CAN 总线协议具有高效、可靠、抗干扰的特点。一般用于汽车，工业自动化，医疗设备

14：ADC（模拟到数字转换器）/DAC（数字到模拟转换器）
ADC分辨率：将模拟信号分成多少个数字值
10位ADC：待变将输入的模拟信号映射为2的10次方，1024个数字值
12位：2的12次方（4096）
分辨率越高，精度越高，表示数字化后的数据更接近真实模拟信号，但是需要更多的计算资源
ADC采样率计算：每秒钟采集多少此模拟信号的数据
一般用于传感器的转换，一般传感器都是模拟信号，输出的是模拟电压或者电流信号


15：PWM(脉宽调制)
用于调整数字信号宽度（占空比）以实现模拟控制的技术


16：Linux驱动开发
字符驱动：按字符方式访问的设备，一般以字节为单位进行数据传输。如：串口，键盘，终端
    open()/read()/write()等
块驱动：数据的传输按照块进行，一般是512字节或者4KB字节，硬盘，山村，SD卡，一般在/dev/sda下
网络驱动：负责管理网络设备接口，支持网络协议栈和硬件的交互。如网卡，wifi

17: TCP
三次握手，面向连接的通信，可靠的。

服务器->客户端发同步确认包，同意连接
客户端->服务器发送连接成功确认包

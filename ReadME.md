pycharm快捷键使用技巧
Ctrl+d 复制当前行、或者选择的块
Ctrl+n 跳转到类
Ctrl+shift+n 快速查找文件名
Ctrl+shift+f 全局查找，快速查找关键字的文件
Ctrl+shift+r 全局替换
Ctrl+鼠标左键 查看源代码
Ctrl+alt+方向左右键 看源码的时候前进返回
Ctrl+a 全选
ctrl+alt+l 格式化代码
Ctrl+/ 注释选中的代码
Alt+enter 自动导入包
Tab 多行同时向后移动
shift+tab 与Tab相反
alt+选中像notepad++多行操作
Ctrl+c 复制当前行或选定的代码块到剪贴板
Ctrl+x 剪切当前行或选定的代码块到剪贴板
Ctrl+v 粘贴
Ctrl+alt+<按键 返回上一个位置
Ctrl+alt+>按键 返回下一个位置
